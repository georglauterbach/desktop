## GTK Themes

The directory `gtk/` provides two GTK themes:

1. `Gruvbox-Material-Dark`: [`Fausto-Korpsvart/Gruvbox-GTK-Theme`](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme), made by @sainnhe and packaged by @korpsvart, ([download](https://www.pling.com/p/1681313))
2. `Everforest-Light`: [`Fausto-Korpsvart/Everforest-GTK-Theme`](https://github.com/Fausto-Korpsvart/Everforest-GTK-Theme), made by @sainnhe and packaged by @korpsvart, ([download](https://www.pling.com/p/1695467))

To use them, copy them into `/usr/share/themes/`.
