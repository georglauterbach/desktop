# GTK Themes

The file `gtk.tar.xz` provides two GTK themes:

1. `Gruvbox-Material-Dark`: [`Fausto-Korpsvart/Gruvbox-GTK-Theme`](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme), made by @sainhe and packaged by @korpsvart, [downloaded](https://www.pling.com/p/1681313) and packaged
2. `Everforest-Light`: [`Fausto-Korpsvart/Everforest-GTK-Theme`](https://github.com/Fausto-Korpsvart/Everforest-GTK-Theme), made by @sainhe and packaged by @korpsvart, [downloaded](https://www.pling.com/p/1695467) and packaged

Unpack the archive and move the sub-directories into `/usr/share/themes/`.
