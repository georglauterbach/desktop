# GTK Themes

The file `gtk.tar.xz` provides two GTK themes:

1. `gruvbox-material-dark`: [`Fausto-Korpsvart/Gruvbox-GTK-Theme`](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme), made by @korpsvart, [downloaded](https://www.pling.com/p/1681313) and packaged
2. `everforest-light`: [`Fausto-Korpsvart/Everforest-GTK-Theme`](https://github.com/Fausto-Korpsvart/Everforest-GTK-Theme), made by @korpsvart, [downloaded](https://www.pling.com/p/1695467) and packaged

Unpack the archive and move the sub-directories into `/usr/share/themes/`.
