# My Linux Desktop

This repository contains the setup for my Linux desktop with all graphical programs that I use on a daily basis.

> [!NOTE]
>
> This repository focuses on GUI applications; head over to [`georglauterbach/hermes`](https://github.com/georglauterbach/hermes) to find the TUI applications that I use on a daily basis.

Packages are installed by the [Nix package manager](https://nixos.org/) with the [_Determinate_ Nix downstream](https://determinate.systems/nix/).
